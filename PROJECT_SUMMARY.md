## 🐛 DEBUGGING TIPS

### If Something's Wrong
1. Check console (F12 → Console tab)
2. Read error message carefully
3. Check .env.local configuration
4. Verify Firebase is running
5. Try rebuilding: `npm run build`

### Common Issues
```
Login doesn't work:
  → Check .env.local has Firebase config
  → Verify email is correct
  → Check Firebase Auth is enabled

Entries not showing:
  → Check Firestore has data
  → Verify security rules allow reads
  → Check network tab in DevTools

Emails not sending:
  → Check functions/.env.local
  → Verify Gmail password (app password)
  → Deploy functions: firebase deploy --only functions
  → Check logs: firebase functions:log

Build fails:
  → Clear cache: rm -rf node_modules dist
  → Reinstall: npm install
  → Check Node version: node -v (v18+)
```
